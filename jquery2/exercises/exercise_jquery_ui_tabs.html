Exercise: jQuery UI

In this exercise, you'll use the jQuery UI plugin to create a tabbed interface.

  1. Start by downloading this starter file.
  2. Open the file in your browser to see the content. You can change it if you like.
  3. Use the jQuery UI Documentation or demos to figure out how to create a tabbed interface.

No solution is provided for this exercise. The code is easy! The tough part can be looking through the documentation to find the answer.
