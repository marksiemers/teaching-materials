Making a Video Player in jQuery

In this exercise, you will use jQuery to create thumbnail video links for youtube videos.

  1. Start by downloading this file.

  2. First, download the jQuery library from jquery.com and save it into the same directory as your exercise files. Then, add a <script> tag to the page which loads in the jQuery library you just downloaded. The script tag will look something like this: 

      <script type="text/javascript" src="PUT THE FILENAME HERE"></script>

  3. Next, follow the code comment prompts - any DOM manipulation can be done with jQuery. If you're not sure how to do something with jQuery, google for "X with jQuery" and you will likely find the jQuery docs or StackOverflow questions.

Make sure that you use your browser developer tools to make debugging easier while working on this. Check for errors, and use console.log() to figure out how far your code makes it, and what the values of your variables are along the way.

  var addVideoThumbnail = function($videoLink) {
    // What jQuery method can we use to get the linkUrl?
    var linkUrl = $videoLink.attr('href'); /* YOUR CODE GOES HERE */

    // We need a thumbnail URL from youtube, how do we get it?
    // See https://dbc-gdi.netlify.com/jquery2/youtube_doc.html
    var thumbnailUrl = youtube.generateThumbnailUrl(linkUrl);/* YOUR CODE GOES HERE */

    // We need a new image element, with the src set to the thumbnailUrl
    // How can we do this using jQuery?
    /* YOUR CODE GOES HERE */
    var $thumbnailImg = $('<img>')
    $thumbnailImg.attr('src', thumbnailUrl);

    // We need to append the thumbnail to the video link.
    // How do we use jQuery to do that?
    $videoLink.append($thumbnailImg);/* YOUR CODE GOES HERE */
  };

  var $videoLinks = $('a');

  // How do we add a thumbnail to all the videos within this for loop?
  for (var i = 0; i < $videoLinks.length; i++) {
    var $videoLink = $($videoLinks[i]);
    addVideoThumbnail($videoLink);
  }
