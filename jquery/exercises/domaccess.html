jQuery: DOM access

The DOM Detective

  • Open up girldevelopit.com in Chrome or Firefox, and open up the console (CMD-Option-I on Macs).
  • Use jQuery to find the following parts of the page. You probably first want to use Inspect Element to figure out what their tag names, classes, and IDs are.
    1. Every image on the page
    2. The navigation area in the upper right
    3. The MailChimp sign-up form in the bottom
    4. The upper left-hand logo that says GDI (Hint: use CSS descendant selectors)
    5. The logos of the media sources that featured Girl Develop It (lifeHacker, TED, etc., at the bottom of the page)
    6. The big heading that says "DON'T BE SHY DEVELOP IT"
    7. BONUS: all the dots in the map


// Every image on the page
$("img");

// The navigation area in the upper right
$(".nav");

// The MailChimp sign-up form in the bottom
$("#mc_embed_signup");

// The upper left-hand logo that says GDI
$(".logo img");
// Or, even better
$(".logo").find('img');

// The logos of the media sources that featured Girl Develop It
$(".press-logos img");
// Or, even better
$(".press-logos').find("img");

// The big heading that says "DON'T BE SHY DEVELOP IT"
$(".opener h1");
// Or, even better
$(".opener").find("h1");

// BONUS: all the dots in the map
$("circle");
// Or, even better
$('.map').find('circle');
